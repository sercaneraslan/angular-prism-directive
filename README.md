angular-prism-directive
=======================

Angular <a href="http://prismjs.com/">Prism</a> Directive

Usage:

- Include prism.js and prism.css
- Add "prism" attribute in the code tag.

Example:

```
<pre>
    <code class="language-javascript" prism>
console.log('Hello World');
   </code>
</pre>
```
